# Gradly — Backend

## Getting Started

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Configure environment variables
A `.env.example` file is included in the repository with the required variable names but no values. Copy it to `.env` and fill in your credentials before running the application.
```bash
cp .env.example .env
```

### 3. Run the application
```bash
python app.py
```
